# INFINITEX Services — Live Admin Dashboard

هذا مشروع صفحات خدمات ما بعد البيع (Static) مع لوحة تحكم **Live** لتعديل المحتوى والوسائط بدون تنزيل/رفع ملف JSON.

## ما الذي يمكن التحكم به؟
- النصوص: عناوين، وصف، نصوص أزرار.
- الروابط: رابط المتجر، رابط سياسة الضمان.
- إعدادات Yamm: baseUrl + hostname + أسماء البراميترز.
- الوسائط: رفع صور من لوحة التحكم واستخدامها داخل المحتوى.

## كيف يعمل فنيًا؟
- يتم تخزين المحتوى والوسائط داخل **Netlify Blobs** عبر Netlify Functions.
- الصفحات تقرأ من `/api/content` وتطبّق القيم على الواجهة، وتحدّث نفسها تلقائيًا.
- رفع الصور يتم عبر `/api/media`.

## التشغيل على Netlify
اتبع التعليمات داخل الملف:
- `NETLIFY_SETUP_README.txt`

## المسارات
- `/admin` لوحة التحكم
- `/api/auth` تسجيل الدخول (JWT)
- `/api/content` قراءة/تحديث المحتوى
- `/api/media` رفع/قراءة الوسائط
- `/api/track` تتبع الطلب
